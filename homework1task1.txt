Linux terminal:

Сценарий:

1)  Посмотреть где я - pwd
2)  Создать папку - mkdir foldername 
3)  Зайти в папку - cd foldername
4)  Создать 3 папки - mkdir foldername1 foldername2 foldername3
5)  Зайти в любую папку - cd fordername2
6)  Создать 5 файлов (3 txt, 2 json) - touch Dad.txt Mother.txt Sister.txt Brother.json Grandmother.json
7)  Создать 3 папки - mkdir newfolder1 newfolder2 newfolder3
8)  Вывести список содержимого папки - ls -la
9)  + Открыть любой txt файл - vim filename.txt
10)  + Написать туда что-нибудь,любой текст - press i - something here i texted 
11)  + Сохранить и выйти - press ESC + press :wq 
12)  Выйти из папки на уровень выше - cd ..
13)  Переместить любые 2 файла, которые вы создали, в любую другую папку - mv foldername1/filename.txt foldername1/filename.json foldername3
14)  Скопировать любые 2 файла, которые вы создали, в любую другую папку - cp fordername1/filename.json fordername3/filename.txt foldername2
15)  Найти файл по имени - find . -name filename.txt or filename.json  ( and so on )
16)  Просмотреть содержимое в реальном времени(команда grep) - cat filename.txt or grep option template filename.txt
17)  Вывести несколько первых строк из текстового файла - head -n filename.txt ( n - количество строк )
18)  Вывести несколько последних строк из текстового файла - tail -n filename.txt ( n - количество строк )
19)  Просмотреть содержимое длинного файла(команда less) - less filename.txt
20)  Вывести дату и время - date
